Game Designer

Job Description

The Game Designer is the designing core of a project of a game, they will document the features of the game including but not limited to: mechanics, storyline, characters, etc. They are also responsible for communicating with the team members so the game produced just like it was designed.

Job Titles:

  • Design Director
  • Designer
  • Level Designer
  • World Designer

Tasks

  1. Balance and adjust gameplay experience to ensure the critical and commercial success of the product
  2. Provide feedback to designers and other colleagues regarding game design features
  3. Create core game features, including storylines, role-playing, mechanics, and character biographies for a new video game or franchise
  4. Devise missions, challenges, or puzzles to be encountered in gameplay
  5. Guide design discussions between development teams
